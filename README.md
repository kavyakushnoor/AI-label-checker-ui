# AI Beverage Label Checker UI

A cloud-based beverage label validation application that uses OCR and automated label analysis to process beverage packaging images and return validation results in real time.

This project provides a lightweight frontend interface for uploading beverage label images and displaying analysis results generated by an AI-powered backend service.

---

## Overview

The application allows users to:

* Upload beverage label images through a web interface
* Extract label text using OCR processing
* Validate beverage label content using backend analysis logic
* Receive structured validation results and status feedback
* Interact with a serverless cloud-native architecture

## Features

### Image Upload Workflow

Users can upload beverage label images directly from the browser.

### OCR Processing

Uploaded images are processed through Google Vision AI to extract text content from beverage labels.

### Automated Validation

Extracted label information is evaluated using backend validation logic and returned as structured output.

### Responsive UI

Lightweight frontend built with React + Vite for fast loading and quick interaction.

### Cloud-Native Deployment

Frontend and backend are independently deployed for scalability and simplified maintenance.

## Local Development Setup

### Prerequisites

* Node.js 18+
* npm or yarn

### Clone Repository

```bash
git clone <repo-url>
cd AI-label-checker-ui
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Application runs locally:

```text
http://localhost:5173
```

---

## Environment Variables

Create a `.env` file:

```env
VITE_API_BASE_URL=<backend-api-url>
```

Example:

```env
VITE_API_BASE_URL=https://your-backend-url
```
